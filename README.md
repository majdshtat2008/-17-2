# موقع رومانسي لِـ مجد الدين و بيسان

موقع صغير مصمم بلغة HTML و CSS و JavaScript لعرض صورتكما، تاريخ التعارف، ومغنيكما المفضلين.

كيفية الاستخدام
- ضع صوركما في المجلد `images/` باسمين `majd.jpg` و `bisan.jpg` (يمكن أن تكون png).
- افتح `index.html` في المتصفح.

ملفات المشروع
- `index.html` — الصفحة الرئيسية
- `styles.css` — أنماط التصميم
- `script.js` — تفاعلات بسيطة (زر رسالة حب، تأثير قلوب)

معلومة إضافية
- يوجد زر تبديل اللغة أعلى الصفحة يمكّنك من عرض الموقع بالعربية أو بالإنجليزية.

- يوجد قسم "رسالتك الخاصة" يحوي محرر نصي لحفظ رسالتك محلياً وتنزيلها كملف نصي.

- أضفت رسالة طويلة افتراضية كتبتها خصيصاً لكِ (مجد الدين) موجودة داخل محرر الرسائل عند أول فتح للموقع؛ تستطيع تعديلها، حفظها، أو تنزيلها.

ملاحظات
- قمت بإضافة خلفية غروب/سماء متحركة لتعطي إحساس "السماء اشتعلت" كما طلبت.
- إن أردت أضيف مشغل أغاني أو ترجمة إنجليزية أو أي تغييرات تصميمية أخرى.
